DevOps - это не только методология с набором практик, а целая кульура взаимодействия та активного сотрудничества со специалистами одной категории со специалистами другой категории(например: разработчики с информационно-технологическому обслуживаннию). Термин "DevOps" многобразен. В термин "DevOps" входят такие професии:QA;OPS;Build Engineer;Infrastructure Engineer; Release Engineer.  
DevOps is not only a methodology with a set of practices, but a whole culture of interaction and active cooperation between specialists of one category and specialists of another category (for example: IT developers). The term "DevOps" is versatile. The term "DevOps" includes the following professions:QA;OPS;Build Engineer;Infrastructure Engineer; Release Engineer.
